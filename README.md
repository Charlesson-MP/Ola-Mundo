## Tipos de listas
### Lista de tópicos
* __Item 1__
   - _Subitem 1.1_
   - _Subitem 1.2_
* __Item 2__
   - _Subitem 2.1_
   - _Subitem 2.2_
### Lista Enumerada
1. __Primeiro item__
   1. _Subitem 1.1_
   2. _Subitem 2.1_
2. __Segundo item__
   1. _Subitem 2.1_
   2. _Subitem 2.2_
### __Lista de tarefas__
   - [x] __Tarefa 1__
      - [x] _Subtarefa 1.1_
      - [x] _Subtarefa 1.2_
   - [ ] __Tarefa 2__
      - [x] _Subtarefa 2.1_
      - [ ] _Subtarefa 2.2_
